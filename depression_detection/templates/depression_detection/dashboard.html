About the System

This system uses Machine Learning and Deep Learning techniques to detect depression using EEG signals. The system implements SVM and CNN algorithms for accurate classification of Normal and Depressed individuals.

  • SVM - Support Vector Machine for baseline classification
  • CNN - Convolutional Neural Network for deep learning-based prediction
  • Performance evaluated using Accuracy, Precision, Recall, and F1-Score
